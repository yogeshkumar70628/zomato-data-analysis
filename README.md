
# Zomato Restaurant Data Analysis

##  Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a dataset of Indian restaurants.
The objective is to analyze restaurant data and extract meaningful insights such as ratings, locations, pricing patterns, and customer trends.

Using Python data analysis libraries, the dataset is cleaned, processed, and visualized to better understand the restaurant ecosystem.

---

##  Objectives

* Explore and understand the dataset
* Perform **data cleaning and preprocessing**
* Handle **missing values and duplicates**
* Analyze restaurant ratings and pricing
* Visualize insights using charts and graphs

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

##  Dataset

The dataset includes information about restaurants such as:

* Restaurant Name
* Location
* Address
* Ratings
* Average Cost
* Cuisine Types
* Online Order Availability

---

##  Data Analysis Process

1. Import Python libraries
2. Load the dataset
3. Explore dataset structure
4. Clean the dataset
5. Handle missing values
6. Perform data visualization
7. Extract meaningful insights

---

## Insights Generated

* Distribution of restaurant ratings
* Popular restaurant locations
* Cost trends across different areas
* Relationship between ratings and pricing


3. Open the notebook

